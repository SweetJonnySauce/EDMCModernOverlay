

# Multiline text
`overlay.py` from BGS-Tally ( https://github.com/aussig/BGS-Tally/blob/develop/bgstally/overlay.py ) is a great example of multi-line text implementation.

`overlay.py` from Bioscan ( https://github.com/Silarn/EDMC-BioScan/blob/master/src/bio_scan/overlay.py ) is another great example.

# Vector Images
The Bioscan Radar found in https://github.com/Silarn/EDMC-BioScan/blob/master/src/bio_scan/overlay.py is a great example of a vector image with animation.

LandingPad from Bgol is another great example of vector images https://github.com/bgol/LandingPad/blob/master/lpads/starport.py

# Markers
Both Bioscan Radar ( https://github.com/Silarn/EDMC-BioScan/blob/master/src/bio_scan/overlay.py ) and [EDR's](https://github.com/lekeno/edr/tree/master/edr) `navroute_config` in `ingamemsg.py` are excellent implementation of the vect marker

# Define Plugin Groups
`overlay.py` from BGS-Tally version 5.4.0+ ( https://github.com/aussig/BGS-Tally/blob/develop/bgstally/overlay.py ) is a great example of the `define_plugin_group` implementation.
